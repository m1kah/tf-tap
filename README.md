# M1kah Tf-tap

## How do I install these formulae?

`brew install m1kah/tf-tap/<formula>`

Or `brew tap m1kah/tf-tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "m1kah/tf-tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
